1. **Create New Project**
   - Klik "New Project"
   - Pilih "Deploy from GitHub repo"
   - Pilih repository Anda

2. **Railway Auto-Detect Laravel**
   - Railway akan otomatis detect Laravel
   - Setup build command otomatis

3. **Add PostgreSQL Database**
   - Klik "New" â†’ "Database" â†’ "PostgreSQL"
   - Railway akan auto-setup connection string

4. **Setup Environment Variables**
   ```
   APP_NAME="Managemen Data Santri"
   APP_ENV=production
   APP_DEBUG=false
   APP_URL=https://your-app.railway.app
   
   DB_CONNECTION=pgsql
   DB_HOST=${{Postgres.PGHOST}}
   DB_PORT=${{Postgres.PGPORT}}
   DB_DATABASE=${{Postgres.PGDATABASE}}
   DB_USERNAME=${{Postgres.PGUSER}}
   DB_PASSWORD=${{Postgres.PGPASSWORD}}
   ```

5. **Generate APP_KEY**
   - Buka terminal di Railway
   - Run: `php artisan key:generate`

6. **Run Migration**
   - Run: `php artisan migrate --force`

7. **Setup Storage Link**
   - Run: `php artisan storage:link`

8. **Deploy!**
   - Railway akan auto-deploy setiap push ke GitHub
