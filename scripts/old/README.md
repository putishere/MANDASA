# scripts/old

Script perbaikan/temporary yang sudah tidak digunakan.

Folder ini berisi file-file script yang dibuat untuk troubleshooting dan perbaikan aplikasi. File-file ini sudah tidak diperlukan lagi setelah aplikasi stabil.

## File di Folder Ini

- `fix_*.php` - Script perbaikan
- `clear_*.php` - Script clear cache/session
- `create_*.php` - Script create data
- `test_*.php` - Script test
- `view_*.php` - Script view database
- `setup_*.php` - Script setup
- `add_*.php` - Script add data
- `jalankan_*.php` - Script jalankan migration
- Dan lainnya...

## Catatan

File-file ini bisa dihapus jika sudah tidak diperlukan lagi. File backup tersimpan di folder `backup/scripts/`.
